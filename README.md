# Telecom Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn behavior in a telecommunications company using Power BI.

The dashboard identifies churn patterns, revenue impact, customer demographics, and key reasons for customer attrition.

## Tools Used

- Power BI
- DAX
- Power Query

## Key Metrics

- Total Customers: 7,043
- Churn Rate: 26.54%
- Revenue Lost: $4M

## Key Insights

- Customers on month-to-month contracts showed the highest churn rates.
- Competitor offers were the leading reason for customer loss.
- Fiber optic customers represented a large proportion of churned customers.
- Churn generated significant revenue loss for the business.

## Dashboard Features

- Executive KPI Summary
- Revenue Impact Analysis
- Customer Segmentation
- Churn Reason Analysis
- Contract Type Comparison
